Temat: SQL - Podstawowe polecenia


  1. Podstawowe polecenia sql

    1. Polecenie CREATE


      Sluzy do tworzenia uzytkownikow/tabel/baz danych

      • Uzytkownik

        --tworzenie uzytkownika
CREATE USER 'nazwa_uzytkownika'@'localhost' IDENTIFIED WITH mysql_native_password BY 'haslo';

      • Baza danych

        --tworzenie bazy
CREATE DATABASE nazwa_bazy;

      • Baza danych

        --tworzenie tabeli
CREATE TABLE nazwa_tabeli(
    typ_kolumny1 nazwa_kolumny1,
    typ_kolumny2 nazwa_kolumny2,
);

    2. Polecenie DROP


      Sluzy do usuwania tabeli/uzytkownika/bazy danych

      • Uzytkownik

        --usuwanie uzytkownika
DROP USER 'nazwa_uzytkownika'@'localhost';

      • Baza danych

        --usuwanie bazy
DROP DATABASE nazwa_bazy;

      • Baza danych

        --usuwanie tabeli
DROP TABLE nazwa_tabeli

    3. Polecenie ALTER


      Sluzy do modyfikacji tabeli

      • Dodawanie kolumny

        --dodawanie kolumny 
ALTER TABLE nazwa_tabeli ADD nazwa_kolumny typ_kolumny

      • Usuwanie kolumny

        --usuwanie kolumny 
ALTER TABLE nazwa_tabeli DROP COLUMN nazwa_kolumny;

      • Zmiana nazwy kolumny

        --zmiana nazwy kolumny 
ALTER TABLE nazwa_tabeli COLUMN RENAME stara_nazwa to nowa_nazwa

      • Zmiana typu kolumny

        --zmiana typu kolumny 
ALTER TABLE nazwa_tabeli MODIFY COLUMN nazwa_kolumny typ_danych

    4. Polecenie UPDATE


      Update sluzy do modyfikacji wpisanych danych w tabeli 
      --zmiana danych w tabeli
UPDATE nazwa_tabeli SET nazwa_kolumny="wartosc" WHERE warunek
