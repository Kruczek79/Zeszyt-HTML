Temat: React wprowadzenie

  1. Czym jest react


    React to rozbudowana javascriptowa biblioteka służąca do tworzenia interfejsów użytkownika.
    Oparta jest na odizolowanych komponentach tworzacych cala aplikacje
    Komponenty maja stan ktorym mozna zarzadzac
    Uzywa sie w nim JSX czyli znacznikow html w javascripcie

  2. Instalowanie reacta


    Aby kozystac z reacta trzeba go zainstalowac.
    Istnieja 3 rozne sposoby na zainstalowanie reacta

    1. Create react app


      Instalujemy Node JS na komputerze oraz wpisujemy polececnie w wierszu polecen: 
      
    npx create-react-app "nazwa projektu" //instaluje nam reacta i tworzy projekt

    2. Vite


      Instalujemy Node JS na komputerze oraz wpisujemy polececnie w wierszu polecen: 
      
    npm create vite@latest //i postepujemy dalej zgodnie z dalszymi instrukcjami

    3. CDN


      Zalaczamy JS przez CDN(content delivery network) za pom znacznika script w html 
      <script crossorigin src="https://unpkg.com/react@18/umd/react.development.js"></script>
<script crossorigin src="https://unpkg.com/react-dom@18/umd/react-dom.development.js"></script>
<script src="https://unpkg.com/@babel/standalone/babel.min.js"></script>
