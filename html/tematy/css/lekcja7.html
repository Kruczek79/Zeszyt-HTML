Temat: Transitions transform

  1. Transform


    Transform to wlasciwosc pozwalajaca na przerozne transformacje elementow

    1. Translate


      Translate sluzy do przesuwania elementow (przesuniecie x, przesuniecie y) 
      div{
    transform: translate(20px, 20px)
}

    2. Scale


      Scale sluzy do skalowania elementow 
      div{
    transform: scale(2)
}

    3. Rotate


      Rotate sluzy do obracania elementow 
      div{
    transform: roatate(40deg)
}

  2. Transitions


    Transitions to sposob na uplynnienie zmiany wlasciwosci aby powstala taka jakby plynna animacja (co ma uplynnic, czas, rodzaj animacji) 
    div{
    transition: all 0.5s ease;
}
