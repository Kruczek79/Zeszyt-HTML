Temat: Box Model

  1. Co to box model?


    Box model to margines, padding, obramowanie ktore dziala razem ze soba tworzac element

  2. Elementy Box modelu CSS

    1. Margin - margines


      Aby ustawic margines dla elementu ustawiamy wlasciwosc margin-strona: wartosc 
                                          p.margin{
    /*top - gora, bottom - dol*/
    /*right - prawo, left - lewo*/
    margin:2px;
    margin-left:20px;
}

    2. Padding


      Padding rozszerza nam element w kazda strone
      Aby nadac padding uzywamy padding-strona: wartosc 
                                          p.padding{
    /*top - gora, bottom - dol*/
    /*right - prawo, left - lewo*/
    padding:2px;
    padding-top:20px;
}

    3. Border - obramowanie


      Obramowanie obramowuje nam element na dany kolor
      Aby nadac obramowanie uzywamy border-strona: wielkosc rodzaj kolor       
                                          p.border{
    /*top - gora, bottom - dol*/
    /*right - prawo, left - lewo*/
    border:2px solid black;
    border-right:2px solid black;
}

  3. Rodzaje obramowania

    1. Solid


      Podstawowe obramowanie nie przerywane

      Lorem, ipsum.

                                          p.solid{
    border:2px solid #fffffff;            
}

    2. Dashed


      Przerywane obramowanie malymi liniami

      Lorem, ipsum.

                                          p.dashed{
    border:2px dashed #fffffff;            
}

    3. Dotted


      Przerywane obramowanie kropkami

      Lorem, ipsum.

                                          p.dotted{
    border:2px dotted #fffffff;            
}

    4. Double


      Podwojne obramowanie

      Lorem, ipsum.

                                          p.double{
    border:2px double #fffffff;            
}
