Temat: CSS - wprowadzenie

  1. Sposoby umieszczania css

    1. Inline


      CSS mozemy umiescic tzw inline czyli za pom atrybutu html style 
                                          <a style="background-color:red">linkacz czerwony</a>

    2. Za pom znacznika style


      CSS mozemy umiescic w sekcji head w znaczniku style 
                                          <head>
    <style>
        <!--umieszczamy tu style css-->
    </style>
</head>

    3. Zewnetrzny plik .css


      CSS mozemy umiescic w oddzielnym pliku css
      Zalaczamy go z uzyciem znacznika link w sekcji head       
                                          <head>
    <link rel="stylesheet" href="sciezka_do_pliku.css">
</head>

  2. Selektory CSS i reguly styli


    1. Reguly styli


      W css tworzy sie tzw reguly styli(ang: rulesets),
      ktore okreslaja wyglad i polozenie elementu html na stronie

    2. Selektory

      • Selektory


        Do okreslenia reguly stylu uzywamy selektorow,
        ktore okreslaja jaki element chcemy wystylizowac

      • Budowa selektora

        :

  3. ID oraz Klasa

    1. Klasa


      Klasa to selektor dla wiecej niz jednej ilosci elementow.
      Nadajemy ja uzywajac atrybutu html class       
                                          <div class="klasa"></div>
      Chcac ustawic styl dla klasy uzywamy selektora ".nazwa_klasy" 
                                          .nazwa_klasy{
    color:red;                                
}

    2. ID


      ID to selektor ktory identyfikuje pojedynczy element html
      Nadajemy go uzywajac atrybutu html id       
                                          <div id="identyfikator"></div>
      Chcac ustawic styl dla elementu z id uzywamy selektora "#identyfikator" 
                                          #identyfikator{
    color:red;                                
}
