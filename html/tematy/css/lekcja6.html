Temat: Psuedo klasy CSS

  1. Co to pseudo klasy?


    Pseudo klasy to selektory css dla specjalnych zdarzen na elemencie (np: najechanie na element myszka)

  2. Przyklady pseudo klas

    1. najechany link

      a:hover{
    color:red;                                
}
      najedz mnie

    2. odwiedzony link

      a:visited{
    color:green;                                
}
      odwiedz mnie

    3. wybrany link

      a:active{
    color:blue;                                
}
      aktywny jestem

    4. pierwsze dziecko

      p:first-child{
    color:blue;                                
}
      Jestem kolorowym dzieckiem

  3. Co to pseudo elementy?


    Pseudoelement CSS służy do stylizowania określonych części elementu
    (np.: zmiana wyglądu pierwszej litery lub
    linijki elementu wstawianie zawartości przed lub po elemencie )

    1. Pierwsza literka

      div::first-letter{
    color:aqua;
    font-size:50px;                                
}
      J estem duza literka

    2. After

      div::after{
    content:"after";
    color:red;                              
}
      tekst

    3. Before

      div::before{
    content:"before";
    color:blue;                              
}
      tekst

    4. Selection

      div:selection{
    background-color:orange;
    color:red;                              
}
      kolorowa selekcja
