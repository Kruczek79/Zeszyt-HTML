Temat: FlexBox

  1. FlexBox co to


    FlexBox to sposob rozmieszczania elementow na stronie.
    Aby uzyc flex boxa trzeba stworzyc element,
    ktory bedzie posiadal wlasciwosc display:flex;
    Nastepnie uzywamy innych wlasciwosci flexBoxa do rozmieszczenia elementow.

  2. Jak uzyc flex boxa

    1. Display: flex


      Nadajemy elementowi flexboxa. 
                                          div.container{
    display:flex;
}

    2. Flex-direction


      Flex box posiada 2 kierunki podstawowy i poboczny.
      Domyslnie kierunek podstawowy to row
      tzn ze elementy we flexboxie beda ustawiac sie w rzedzie,
      a gdy jest column to elementy beda ustawiac sie pod soba,
      Mozna takze uzyc row-reverse lub column-reverse co ulozy nam elementy na odwrot do kierunku       
                                          div.container{
    display:flex;
    flex-direction:row/column;                                      
}

    3. Justify Content


      Justify content ustawia nam elementy wzgledem podstawowego kierunku flex (domyslnie row)
      Przyjmuje wartosci center, flex-start,flex-end,space-around,space-between       
                                          div.container{
    display:flex;
    /*center(srodek), flex-start(na poczatku elementu flex), */
    /*flex-end(na koncu elementu flex),space-around(odstep do okola),*/
    /*space-between(odstep miedzy elementami)*/
    justify-content: center;
}

    4. Align Items


      Align Items ustawia nam elementy wzgledem dodatkowego kierunku flex (domyslnie column)
      Przyjmuje wartosci center, flex-start,flex-end,space-around,space-between       
                                          div.container{
    display:flex;
    /*center(srodek), flex-start(na poczatku elementu flex), */
    /*flex-end(na koncu elementu flex),space-around(odstep do okola),*/
    /*space-between(odstep miedzy elementami)*/
    align-items:center;
}
