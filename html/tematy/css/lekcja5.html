Temat: Position

  1. Co to jest Position


    Position okresla jak dany element ma byc pozycjonowany na stronie.

  2. Wartosci Wlasciwosci Position

    1. static


      Pozycja statyczna(domyslna), element jest pozycjonowany normalnie w dokumencie 
                                          p{
    position:static;                                    
}

    2. relative


      Pozycja relatywna, element jest pozycjonowany wzgledem swojego domyslnego polozenia w dokumencie 
                                          p{
    position:relative;                                    
}

    3. absolute


      Pozycja Absolutna, element jest wyswietlany wzgledem calego dokumentu
      (lub rodzica ktory posiada pozycje relatywna)
      nad wszystkim innym przez co moze nachodzic na inne elementy       
                                          p{
    position:absolute;                                    
}

    4. fixed


      Dziala jak sticky ale wyswietlany jest jak pozycja absolutna 
                                          p{
    position:fixed;                                    
}

    5. sticky


      Element jest pozycjonowany przyklejonie do dokumentu (lub rodzica) gdy scrollujemy strone element zawsze pozostaje na swoim miejscu. 
                                          p{
    position:sticky;                                    
}

  3. Nowe wlasciwosci


    Gdy element ma ktoras z pozycji(oprocz static) otrzymuje dostep do nowych wlasciwosci

    1. Top, bottom, right, left

      Okresla jak ma byc pozycjonowany element ile przesuniecia wzgledem ustawionej pozycji. 
      p{
    position:relative;
    top:10px;
    left:30px;
    right:50px;
    bottom:20px;                                
}

    2. Z-index


      Okresla kolejnosc elementu wzgledem reszty (im wiecej tym bedzie wyzej). 
      p{
    position:relative;
    z-index:1;                                
}
