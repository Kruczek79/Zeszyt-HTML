Temat: Animacje CSS

  1. Animacje


    Aby dodac animacje do elementu uzywamy wlasciwosci animation: nazwa_animacji czas 
    div{
    animation: animacja 1s;
}

    1. Definicja animacji


      Aby zdefiniowac animacje uzywamy funkcji @keyframes nazwa_animacji i
      ustawiamy co ma sie zdarzyc z elementem w poszczegolnym procencie animacji       
      div{
    animation:ruch 2s;
}                                    

@keyframes ruch{
    10%{
        transform:translate(10px, 20px);
    }
    50%{
        transform:translate(30px, 40px);
    }
}
