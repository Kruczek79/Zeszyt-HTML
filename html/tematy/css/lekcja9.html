Temat: Grid

  1. Grid


    Grid to sposob na rozmieszczenie elementow na stronie w siatce.
    Jest bardzo podobny do flexboxa lecz jest troche bardziej skaplikowany
    Definiujemy go przy uzyciu display:grid;     
    .grid-container{
    display:grid;
}

    1. grid-template-columns


      Ustawia nam szerokosci kolumn w siatce (1kolumna, 2kolumna, 3kolumna, itd) 
      .grid-box{
    display:grid;
    grid-template-columns: 100px, 100px, 100px;
}
      Funkcja repeat pozwala nam ustawienie tej samej wartosci dla x kolumn (x, wartosc) 
      .grid-box{
    display:grid;
    grid-template-columns: repeat(3, 100px);
}

    2. grid-template-rows


      Ustawia nam szerokosci wierszy w siatce (1wiersz, 2wiersz, 3wiersz, itd) 
      .grid-box{
    display:grid;
    grid-template-rows: 100px, 100px, 100px;
}
      Funkcja repeat pozwala nam ustawienie tej samej wartosci dla x wierszy (x, wartosc) 
      .grid-box{
    display:grid;
    grid-template-rows: repeat(3, 100px);
}

    3. grid-auto-rows


      Ustawia nam szerokosci wierszy w siatce ktore nie dotycza grid-template-rows 
      .grid-box{
    display:grid;
    grid-auto-rows: 100px;
}

    4. grid-auto-columns


      Ustawia nam szerokosci kolumn w siatce ktore nie dotycza grid-template-columns 
      .grid-box{
    display:grid;
    grid-auto-columns: 100px;
}
      Funkcja minmax pozwala nam ustawienie minimalnej wielkosci i maksymalnej wielkosci (minimalna, maksymalna) 
      .grid-box{
    display:grid;
    grid-auto-columns: minmax(200px, auto);
}

    5. Gap


      Gap to odstep od elementow 
      .grid-box{
        display:grid;
        gap: 10px;
    }

    6. grid-template-areas


      grid-template-areas pozwala na ulozenie elementow jak sie chce 
      .grid-box{
        display:grid;
        grid-template-areas: 
        "header header" /*na gorze bedzie element o nazwie header */
        "menu content"  /*z boku bedzie menu a obok content */
        "menu footer"  /*na dole bedzie menu i stopka */
    }
    .grid-item1{
        grid-area:header; /*nadanie nazw elementow grid */
    }    
    .grid-item2{
        grid-area:menu;
    } 
    .grid-item3{
        grid-area:content;
    }      
    .grid-item4{
        grid-area:footer;
    }
      header
      menu
      content
      footer

    7. grid-column


      grid-column pozwala ustawic wielkosc elementu 
      .grid-item{
        grid-column: 1 / 4; /* od 1 kolumny do 4 kolumny */
    }

    8. grid-row


      grid-row pozwala ustawic wielkosc elementu 
      .grid-item{
        grid-row: 1 / 4; /* od 1 rzedu do 4 rzedu */
    }
