Temat: CSS - kolory

  1. Kolory


    W cssie kolory ustawiamy na 3 sposoby:

    1. RGB


      Ustawiamy wartosci dla koloru
      R (czerwony) G (zielony) B (niebieski) od 0 - 255 
                                          p{
    color: rgb(255, 0, 0);
}

    2. Hex


      Ustawiamy hexadecymalny(szestnastkowy) kod koloru poprzedzany przez "#"
      od 0-1 i od A-F gdzie fff to bialy a 000 to czarny       
                                          p{
    color: #ff0000;
}

    3. Nazwa koloru


      Nazwa koloru po angielsku np:. red 
                                          p{
    color: red;
}

  2. Background-color i Color


    W CSS mozemy ustawiac kolor tekstu i kolor tla za pom:

    1. Background-color


      Ustawia to kolor tla za elementem 
                                          p{
    background-color: red;
}

    2. Color


      Ustawia to kolor tekstu w elemencie 
                                          p{
    color: red;
}
