Temat: Obiektowy JS - Obiekty

  1. Co to obiekt


    Obiekt – wyodrębniony element rzeczywistości, element może być materialny lub abstrakcyjny
    Obiekt posiada wlasciwosci(pola) i funkcje(metody)
    Np: Obiekt programista
    wlasciwosci: kolorWlosow, jezykiProgramowania, imie, nazwisko
    Metody: programuj(), uczSieProgramowania()

  2. Definicja obiektu

    let programista = {
    kolorWlosow: 'zielony',
    jezykiProgramowania: [c++,Java,Js,python],
    imie: 'aleksik',
    nazwisko: 'siur',
    programuj : function(){//kod},                           
}

  3. Odczytanie pol obiektu

    Aby odczytac pole obiektu uzywamy operatora kropki 
    programista.kolorWlosow;

  4. Wywolanie metody

    Aby wywolac metode obiektu uzywamy operatora kropki 
    programista.programuj();
