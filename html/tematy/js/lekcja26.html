Temat: Operatory and(&&) i or(||)

  1. && And


    Sprawdza czy 1 wartosc jest true jesli tak to sprawdza kolejna
    dopoki jakas wartosc nie bedzie false a jesli wartosci sa true to zwraca ta ktora jest ostatnia     
    true && 0 //zwraca 0

    0 && 'makapaka' //zwraca 0

    'makapaka' && 'javascript' //zwraca 'javascript'
    Wszystkie wartosci sa true oprocz (null, NaN, 0, '', undefined)

  2. || OR


    Sprawdza czy jakakolwiek z wartosci jest true i ja zwraca 
    true || 0 //zwraca 0

    0 || 'makapaka' //zwraca 'makapaka'

    'makapaka' || '' //zwraca 'makapaka'
