Temat: DOM HTML

  1. Co to DOM


    DOM (Document Object Model) jest to drzewo reprezentujace
    zaleznosci(hierarchie) miedzy elementami html

  2. Pobieranie elementow w DOMIE


    Pobrac element w js mozna na kilka roznych sposobow.
    Robimy to poto aby manipulowac elementem od strony javascriptu. Oto one:

    1. getElementById

      Pobiera element za pomoca id 
      document.getElementById("id")

    2. querySelector

      Pobiera element za pomoca id lub klasy 
      document.querySelector("#id") //pobiera za pom id
document.querySelector(".klasa") //pobiera za pom klasy

    3. querySelectorAll

      Pobiera wszystkie elementy od danym id lub klasie.
      Zwraca w postaci tablicy elementow       
      document.querySelectorAll(".id"); //pobiera za pom id
document.querySelectorAll(".class"); //pobiera za pom klasy
document.querySelectorAll(.class)[0]; //pierwszy element klasy "class"

  3. Manipulacja elementami w DOMie

    1. innerHTML

      Zmienia wewnetrzny html elementu 
      document.querySelector("#id").innerHTML = "wewnetrzny html elementu";

    2. textContent

      Zmienia tekst elementu 
      document.querySelector("#id").textContent = "wewnetrzny tekst";

    3. style

      Zmienia styl elementu w css 
      document.querySelector("#id").style.color = "red"; //kolor tekstu czerwony

    4. setAttribute

      Ustawia Atrybut html na elemencie 
      document.querySelector("#id").setAttribute("id", "btn"); //ustawia atrybut html id na "btn"

    5. createElement

      Tworzy element html 
      let div = document.createElement("div"); //tworzy element

    6. appendChild

      Dodaje element do potomka elementu 
      let div = document.createElement("div"); //tworzy element
document.querySelector(".content").appendChild(div); //dodaje diva do elementu .content jako potomek


      <div class="content">
    <div></div> <!--to jest ten div utworzony i jest on potomkiem diva .content -->
</div>

    7. removeChild

      Usuwa element potomek elementu 
      let div = document.createElement("div"); //tworzy element
document.querySelector(".content").removeChild(div); //usuwa diva z elementu .content
