    5. Obiekt date


      Reprezentuje czas. Tworzymy go z uzyciem new Date() 
      const data = new Date()//tworzenie obiektu date
      Najwazniejsze metody obiektu date:
      1. getDate()

        Zwraca dzien miesiaca 1-31 
        data.getDate()//1-31
      2. getFullYear()

        Zwraca rok 
        data.getFullYear()//2023
      3. getHours()

        Zwraca godzine 0-23 
        data.getHours()//0-23
      4. getMinutes()

        Zwraca minuty 0-59 
        data.getMinutes()//0-59
      5. getSeconds()

        Zwraca sekundy 0-59 
        data.getSeconds()//0-59
      6. getMonth()

        Zwraca miesiac 0-11 
        data.getMonth()//0-11
