  1. Co to instrukcja warunkowa


    Instrukcja warunkowa to konstrukcja ktora umozliwia wykonanie instrukcji
    gdy dane wyrazenie logiczne jest prawdziwe.
    Wyrozniamy 2 instrukcje warunkowe

    1. Switch


      Switch sluzy do podjemowania decyzji warunkowych.
      Pozwala na krotszy zapis wielu instrukcji if       
                                              switch(warunek)
{
    case przypadek:
        //operacja
        break;//konczy switch
    case przypadek:
        //operacja
        break;//konczy switch
}

    2. If


      If sluzy do podjemowania decyzji dla specyficznych przypadkow. 
                                              if(warunek){
    //operacja                                            
}
