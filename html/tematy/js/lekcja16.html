Temat: Klawisze

  1. Obsluga klawiszy


    Do obslugi klikniec na klawiaturze sluza 3 eventy

    1. Key down


      Kiedy klawisz jest nacisniety i trzymany 
                                          document.addEventListener('keydown', (event) => {
    //kod
}

    2. Key press


      Kiedy klawisz jest nacisniety i trzymany 
                                          document.addEventListener('keypress', (event) => {
    //kod
}

    3. Key up


      Kiedy klawisz jest wypuszczony 
                                          document.addEventListener('keyup', (event) => {
    //kod
}

  2. Przechwycenie nacisnietego klawisza


    Aby przechwycic klawisz uzywamy parametru event w funkcji i wlasciwosci key i code 
                                document.addEventListener('keyup', (event) => {
    event.key; //klawisz nacisniety
    event.code; //kod klawisza nacisnietego
}
