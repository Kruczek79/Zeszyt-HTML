  1. Co to JSON


    JSON to javascript object notation
    Sluzy do transportowania danych

  2. Syntax Jsona


    JSON ma jeden glowny obiekt w ktorym przechowujemy inne obiekty, tablice, stringi i liczby 
    {
    "string" : "makapaka",
    "liczba" : 12,
    "tablica" : [1,2,3,4],
    "bool" : true,
    "obiekt" : {}
}

  3. Parsowanie i zamienianie JSONa


    Aby zamienic javascriptowy obiekt na obiekt jsona uzywamy funkcji JSON.stringify(obiekt)
    Aby zamienic obiekt jsona spowrotem na obiekt javascriptowy to uzywamy funkcji JSON.parse("string jsona")     
    const obiekt = {
    name: "makapaka",
    age: 12    
}
const jsonString = JSON.stringify(obiekt) //'{"name":"makapaka","age":12}' (obiekt jsona);

JSON.parse(jsonString); //{name: "makapaka",age: 12}(obiekt javascriptowy)
