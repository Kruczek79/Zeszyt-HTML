Temat: Tablice

  1. Czym sa tablice


    Tablice to zbior zmiennych.
    Jej poszczególne elementy są rozróżniane poprzez przypisane im liczby nazywane indeksami.
    Indeksy tablic zawsze zaczynają się od zera.

  2. Tworzenie tablic w JS

    Tablice moze wypelniac kazdy rodzaj zmiennej i obiekty 
    let tablica = [] //tworzenie pustej tablicy

    let tablica = [1,2,3,4] //tworzenie tablicy z liczbami

    let tablica = []; //tworzenie pustej tablicy
tablica[0] = "a"; //definiowanie wartosci tablicy o indeksie 0
tablica[1] = "b"; //definiowanie wartosci tablicy o indeksie 1

  3. Metody Tablic

    Tablice to obiekty
    Obiekty posiadaja swoje funkcje(metody)
    Metody tablic:

    1. Map


      Przechodzi przez kazdy element tablicy i robi cos dla kazdego elementu tablicy.
      Zwraca nowa zmieniona tablice       
      const array1 = [1, 4, 9, 16];
const map1 = array1.map(x => x * 2); // [2, 8, 18, 32]

    2. For each


      Przechodzi przez kazdy element tablicy i robi cos dla kazdego elementu tablicy.
      Zwraca zmieniona tablice       
      const array1 = ['a', 'b', 'c'];
array1.forEach(element => console.log(element)); //a b c

    3. Sort


      Sortuje elementy tablicy 
      const array1 = [1, 30, 4, 21, 100000];
array1.sort(); //[1, 100000, 21, 30, 4]

    4. IndexOf


      Sprawdza index podanego elementu w danej tablicy 
      const array1 = [1, 30, 4, 21, 100000];
array1.indexOf(1); //0

    5. Some


      Sprawdza czy jakis element tablicy spelnia warunek 
      const array = [1, 2, 3, 4, 5];
console.log(array.some( (element) => {
    element % 2 === 0
})); //true;

    6. every


      Sprawdza czy wszystkie elementy tablicy spelniaja warunek 
      const array1 = [1, 30, 39, 29, 10, 13];
console.log(array1.every( (currentValue) => currentValue < 40)) //true

    7. Pop


      Kasuje ostatni element tablicy 
      const plants = ['broccoli', 'cauliflower', 'cabbage', 'kale', 'tomato'];
plants.pop();
console.log(plants) //['broccoli', 'cauliflower', 'cabbage', 'kale']

    8. Push


      Dodaje element do tablicy 
      const animals = ['pigs', 'goats', 'sheep'];
animals.push('cows');
console.log(animals); //["pigs", "goats", "sheep", "cows"]

    9. Reverse


      Odwraca tablice 
      const array1 = ['one', 'two', 'three'];
array1.reverse();
console.log(array1); //["three", "two", "one"]
