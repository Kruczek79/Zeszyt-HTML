Temat: Funkcje JS

  1. Co to funkcje

    Funkcja to wydzielona czesc programu komputerowego wykonujaca jakies operacje.
    Wyrozniamy 3 rodzaje funkcji:

    1. Funkcja normalna


      Funkcja normalna wykonuje operacje zawarte w srodku niej. Aby sie wykonala trzeba ja wywolac 
                                          function nazwa_funkcji(argumenty){
    //operacje
    return f //zwraca wartosc f                                       
}
nazwa_funkcji(argumenty); //wywolanie funkcji

    2. Funkcja Anonimowa


      Dziala jak normalna jedynie rozni sie tym ze nie posiada nazwy 
                                          const makapaka = function (argumenty){
    //operacje 
    return f //zwraca wartosc f                                           
};         
nazwa_funkcji(argumenty); //wywolanie funkcji

    3. Funkcja strzalkowa


      Dziala jak normalna funkcja jedynie rozni sie zapisem i jest anonimowa 
                                          (argumenty) => {
    //operacje
    return f //zwraca wartosc f                                      
}

  2. Jak dzialaja parametry(argumenty)


    W "()" funkcji podaje sie tzw argumenty(nietrzeba).
    Sa to zmienne ktore chcemy podac funkcji abysmy mogli je uzywac wewnatrz jej np:     
    function policz(a,b)
{
    let wynik = a * b; //mnozy argumenty przez siebie
    return wynik; //zwraca wynik                            
}
//a = 1, b = 2
policz(1,2); //wynik = 2
