Temat: Zmienne JS

  1. Co to zmienne


    Zmienne to zarezerwowane szufladki w pamieci ram komputera ktorym przypisujemy wartosci.
    Sluzy do przechowywania jakis wartosci.

  2. Rodzaje zmiennych


    Zmienne dzielimy na wiele rodzajow m.in.:

    1. Liczby calkowite (Inty)


      Liczby calkowite bez przecinka np:. 20 
       let i = 20;

    2. Liczby zmienno przecikowe (Floaty)


      Liczby z przecinkiem np:. 1,5 
      let f = 1,5;

    3. Lancuchy znakow (Stringi)


      Zmienne przechowujace tekst np:. "ala ma kota" 
      let str = "ala ma kota";

    4. Booleany (Prawda/Falsz)


      Zmienne prawda lub falsz 
      let bool = true; 
let boolean2 = false;

    5. NaN (Not A Number)


      Wartosc ktora nie jest liczba 
      let nan = NaN;

    6. Undefined


      Wartosc niezdefiniowana (nie posiada zaden z typow) 
      let ud = undefined;

    7. Null


      Wartosc pusta (nie posiada zaden z typow) 
      let n = null;

  3. Sposoby deklaracji zmiennych

    Zmienne deklarujemy na 3 rozne sposoby:

    1. Let


      Let dziala w bloku tzn ze:.
      gdy let jest zadeklarowany w funckji, instrukcji warunkowej, itp
      To js bedzie widziec ta zmienna jedynie w tej funkcji, instrukcji warunkowej nie poza nia       
      let variable1 = true; //zmienna let

    2. Const


      Const jest to wartosc stala nie zmienna. Dziala w zasiegu takim samym jak let 
      const variable1 = "makapaka"; //stala

    3. Var


      Var dziala w zasiegu funkcji lub innych struktur tzn ze:.
      gdy var jest zadeklarowany w funkcji/instrukcji warunkowej
      to js bedzie widziec ta zmienna jakby dzialala globalnie dla calego dokumentu       
      var variable1 = 23; //zmienna war
