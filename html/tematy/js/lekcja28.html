Temat: Import i export - podzial aplikacji na kilka plikow

  1. Import i export


    Pozwala nam na podzial na kilka plikow poprzez importowanie i eksportowanie elementow(funkcji, zmiennych, obiektow, klas).
    Aby moc uzywac tego nalezy do znacznika script dodac type="module"     
    <script type="module" src="main.js"></script>

  2. Export


    Export sluzy nam na eksportowanie elementow z pliku.
    Mozemy uzywac export przed deklaracja albo po deklaracji     
    export const zmienna = "costam"; //export przed deklaracja
class User{
    constructior(name)
    {
        this.name = name;
    }
}
export {User}; //export po deklaracji w {}

  3. Export default


    Export default sluzy do wyeksportowania 1 rzeczy jako domyslna nie trzeba podawac jej nazwy 
    export default class User {
    constructor(name) {
        this.name = name;
    }
}

  4. Import


    Import sluzy do zaimportowania elementow 
    import {nazwa} from 'plik.js'; //importuje element o nazwie nazwa


    import {nazwa} as name from 'plik.js'; //importuje element o nazwie "nazwa" i zmienia nazwe na "name"


    import Nazwa from 'plik.js'; //importuje element wyeksportowany default o nazwie "Nazwa"
