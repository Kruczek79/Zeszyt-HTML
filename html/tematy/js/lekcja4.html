Temat: Petle JS

  1. Co to petle?


    Petla to konstrukcja umozliwiajaca cykliczne wykonywanie ciagu instrukcji okreslona liczbe razy.
    Wyrozniamy 3 rodzaje petli:.

    1. Petla While


      Wykonuje sie dopoki warunek jest prawdziwy 
      while(warunek){
    //operacje
}

    2. Petla For


      Wykonuje sie dopoki iterator(zmienna petli) nie przekroczy jakies wartosci 
      //i = iterator, i=0 poczatkowa wartosc, i++ zwiekszanie iteracji petli
//petla wykona sie od 0 do 10 (9razy)
for(let i=0;i>10;i++){
    //operacje
}

    3. Set Interval


      Wykonuje sie w nieskonczonosc (mozna ja zatrzymac) co odstep czasu podany w argumencie 
      let id = setInterval(() => {
    //iterator petli
    let i = 0;
    //jezeli i bedzie wieksze od 10 zastopuj petle
    if(i > 10){
        clearInterval(id)//stop petli
    }
}, argument);//interwal co ile czasu ma sie wykonywac petla
