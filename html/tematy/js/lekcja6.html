Temat: Eventy w JS

  1. Czym SÄ… eventy?


    Eventy to zdarzenia ktore dzieja sie elementom html(klikniecie, najechanie myszka)
    Metody dodania zachowania gdy event zajdzie na danym elemencie

    1. Inline


      <button event="kod javascriptu">

    2. element.event()


      element.event = function nazwa_funkcji(){
    //operacje 
}

    3. Add event listener


      Add event listener to nasluchiwacz eventow (najlepszy sposob na uzywanie eventow w js) 
      element.addEventListener("event", function nazwa_funkcji(){
    //operacje                                    
});
