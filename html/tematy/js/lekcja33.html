Temat: Try catch - wylapywanie bledow

  1. Try


    Umieszczamy w tym bloku kod aby js sprobowal go wykonac gdy zwroci blad to wykonuje sie catch 
    try{
    someFunction(); //wykonuje sie funkcja
    throw new Error('tekst bledu'); //wyrzuca blad
}

  2. Catch


    Umieszczamy w tym bloku kod ktory ma sie wykonac gdy try zwroci blad 
    catch(e){
    console.log(e); //wypisuje error w konsoli
}

  3. finally


    Umieszczamy to co ma sie wykonac juz po zlapaniu bledow 
                                finally{
    //kod
}
