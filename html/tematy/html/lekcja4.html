Temat: Podstawowe Znaczniki HTML

  1. Budowa Znacznika HTML



    HTML składa sie z tzw znacznikow ktore definiują elementy na stronie internetowej.
    1. Kazdy znacznik zawiera swoje otwarcie oraz  zamkniecie:

    2. Istnieją rowniez wyjątki od tej reguły ktore nie posiadają zamkniecia np:. znacznik img.

    3. Kazdy znacznik moze posiadac swoje atrybuty np:. znacznik "a" posiada swoj atrybut href,
      ktory wskazuje gdzie przeniesc sie po nacisnieciu linku.

  2. Struktura strony HTML



    1. Deklaracja doctype


      Strona html zaczyna sie od tzw deklaracji doctype ktora jest wymagana 
                                  <!DOCTYPE html>

    2. Znacznik html


      Nastepnie znajduje sie znacznik html ktory jest rowniez wymagany.
      Tutaj umieszczamy wszystkie pozostale znaczniki.       
                                  <html></html>

    3. Sekcja Head


      Sekcja head w ktorej umieszczamy wszystko co chcemy załączyc na naszą strone,
      co chcemy aby odwiedzający strone nie widzieli(pliki css i js)       
                                          <head>
<!-- ustawia kodowanie znakow na utf-8   -->
<meta charset="utf-8">

<!-- ustawia tytuł wyswietlany na gorze przegladarki -->
<title></title>        
</head>

    4. Sekcja Head


      Sekcja body w ktorej umieszczamy zawartosc strony internetowej. 
                                          <body></body>
